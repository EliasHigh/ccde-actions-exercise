# ccde-actions-exercise

Dieses Repository enthält Übungen zu **Git** und **GitHub Actions** im Rahmen des Faches Cloud Computing (4abHWII, HTL Anichstraße).

## Inhalt

- `calculator.py` – Einfacher Taschenrechner mit Addition, Subtraktion und Multiplikation
- `app/main.py` – Hello World Beispiel für GitHub Actions
- `.github/workflows/pipeline.yml` – GitHub Actions Pipeline (Quality Assurance)

## Verwendete Technologien

- Python 3
- GitHub Actions
- Ruff (Linting)
- Bandit (Security Check)
- pip-audit (Vulnerability Audit)

## Ausführen

```bash
python calculator.py
```
